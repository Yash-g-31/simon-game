# Simon Game

A fun and interactive Simon Game built with HTML, CSS, and JavaScript.

## How to Play

1. Press any key to start the game
2. Watch the sequence of colored buttons light up
3. Repeat the sequence by clicking the buttons in the correct order
4. Each level adds one more color to the sequence
5. If you make a mistake, the game ends and your score is displayed

## Features

- Progressive difficulty levels
- Highest score tracking
- Interactive visual feedback
- Responsive design

## Local Setup

```bash
npm install
npm start
```

Then open `http://localhost:3000` in your browser.
